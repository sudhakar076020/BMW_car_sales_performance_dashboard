# 🚗 BMW Car Sales Performance Dashboard

## 📊 Project Overview
This project presents an interactive **Car Sales Performance Dashboard** built using **Power BI**, designed to analyze sales trends, customer preferences, and revenue distribution.

The dashboard provides insights into:
- Regional revenue performance  
- Top-selling car models  
- Fuel type market share  
- Price segment contribution  
- Engine size vs price relationship  
- Transmission preferences (Manual vs Automatic)  

---

## 🎯 Objectives
- Analyze **sales performance across regions**
- Identify **top-performing car models**
- Understand **customer preferences** (fuel type & transmission)
- Evaluate **revenue contribution by price segments**
- Explore **relationship between engine size and price**

---

## 📁 Dataset Description
The dataset includes the following columns:

- Model  
- Year  
- Region  
- Fuel_Type  
- Transmission  
- Engine_Size_L  
- Mileage_KM  
- Price_USD  
- Sales_Volume  

---

## 🔧 Data Preparation
The following calculated fields were created:

- **Revenue** = Price_USD × Sales_Volume  
- **Price Category** (Budget, Mid Range, Premium)  
- **Engine Category** (Small, Medium, Large)  
- **Mileage Category**  

---

## 📊 Dashboard Features

### 🔹 KPI Cards
- Total Cars Sold  
- Total Revenue  
- Average Car Price  
- Total Models  
- Average Sales per Model  

### 🔹 Visualizations
- Revenue by Region (Bar Chart)  
- Top Selling Car Models (Column Chart)  
- Fuel Type Market Share (Pie Chart)  
- Revenue by Price Segment (Stacked Chart)  
- Engine Size vs Price Analysis (Scatter Plot)  
- Manual vs Automatic Sales (Donut Chart)  

---

- Model

## 🎨 Design
The dashboard uses a **BMW M Sport inspired theme**:

## 📌 Key Insights
- Certain regions contribute significantly higher revenue  
- Premium cars generate the highest revenue  
- Fuel type distribution shows balanced customer preferences  
- Engine size has a direct relationship with price  
- Automatic cars are slightly more preferred than manual
